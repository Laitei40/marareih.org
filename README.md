# Mara Language Preservation (MLP)

**Preserve. Standardize. Enable Mara.**

Mara Language Preservation (MLP) is a community-driven initiative focused on safeguarding the **Mara language** by building open digital tools, documentation, and datasets. The goal is to ensure that Mara remains usable, accessible, and sustainable in the modern digital ecosystem.
